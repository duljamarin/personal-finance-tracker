# Expense Tracker

React + Tailwind expense tracker sample app with:
- add / edit / delete expenses
- filter by year
- simple monthly summary chart (SVG-like bars)
- dark mode (toggle)
- export / import CSV
- persistence in localStorage

## Setup

1. `npm install`
2. `npm run dev`

Note: Tailwind is configured via PostCSS and Vite. Tailwind classes work out of the box.
